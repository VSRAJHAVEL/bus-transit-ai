# BUS TRANSIT AI — Coimbatore

AI-powered route optimization system for Coimbatore TNSTC city buses. This system leverages ensemble machine learning models and graph algorithms to provide optimal route predictions and live ETA estimates.

## Features
- **Route Finder:** Optimized pathfinding between 53+ stops.
- **Live ETA:** Real-time arrival predictions based on traffic and historical data.
- **Analytics:** Traffic congestion and route performance visualizations.
- **AI/ML Engine:** Ensemble VotingRegressor (Random Forest + Gradient Boosting).
